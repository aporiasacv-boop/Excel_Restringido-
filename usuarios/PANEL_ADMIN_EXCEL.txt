# Panel Admin en Excel (solo usuario Admin)

Tras entrar como Admin se abre la hoja oculta _PanelAdmin con botones:

  Ver colaboradores | Alta usuario | Baja usuario | Reactivar | Volver al formato

Los operadores normales NO ven esa hoja.

Si fallara el panel: Alt+F8 -> AdministrarUsuarios / AltaUsuario / BajaUsuario

Requisitos: Excel de ESCRITORIO, macros habilitadas, API y tunel activos en TI.
