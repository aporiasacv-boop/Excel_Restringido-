# Panel Admin en Excel (solo usuario Admin)

Tras entrar como Admin aparecen 4 botones arriba a la izquierda en la 1ra hoja:

  Colaboradores  - ver lista sincronizada con servidor TI
  Alta usuario   - nuevo colaborador
  Baja usuario   - desactivar
  Reactivar      - volver a activar

Los operadores normales NO ven esos botones.

Requisitos: Excel de ESCRITORIO, macros habilitadas, API y tunel activos en TI.

Tambien funcionan las macros Alt+F8 (AdministrarUsuarios, etc.).
