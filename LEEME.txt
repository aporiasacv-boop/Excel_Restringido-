PC DE TI - EXCEL NIKZON
=======================

PULL:
  git pull origin main

INSTALAR (solo la primera vez):
  0_INSTALAR_CLOUDFLARE.bat   -> descarga cloudflared

SI FALLA "puerto en uso":
  0_LIBERAR_PUERTOS.bat       -> cierra procesos viejos en 8080 y 8011

CADA DIA (en orden):
  1. 1_API_OLNATURA.bat       Python :8011
  2. 2_TUNEL_OLNATURA.bat     URL publica trycloudflare.com

USUARIOS (PC de TI, opcional):
  Editar: usuarios\USUARIOS.txt + 3_CARGAR_USUARIOS.bat

GESTION DESDE EXCEL (solo Admin, Alt+F8):
  AdministrarUsuarios  - ver lista sincronizada con servidor TI
  AltaUsuario          - dar de alta colaborador
  BajaUsuario          - desactivar (no puede entrar)
  ReactivarUsuario     - volver a activar
  OcultarPanelUsuarios - ocultar hoja _Usuarios

  Ver accesos: MostrarAccesos | 4_VER_ACCESOS_SERVIDOR.bat
  Admin: Admin / Admin123!

INSTALAR EXCEL NIKZON:
  1. Copie los 4 archivos NIKZON*.xlsm a la carpeta entrada\
  2. Cierre Excel
  3. Doble clic en 2_INSTALAR_EXCEL.bat
  4. Salida en LISTOS\ -> subir a OneDrive

Ordenes de venta (otra carpeta Prueba_Ov):
  0_TOKEN_NGROK.bat (1 vez), 1_ORDENES_JAVA.bat, 2_NGROK.bat
